# 🎮 CONTADOR GAMER

Projeto desenvolvido em React Native com Expo.

---

# Preview do APP 

O aplicativo possui:
- Contador de pontos;
- Botão de aumentar
- Botão de diminuir;
- Botão de resetar;
- Interface estilo gamer;

---

# Tecnologias utilizadas

- React Native
- Expo
- JavaScript

---

# Como executar o projeto

## 1. Clonar o repositório
```
git clone https://github.com/LaraCSiqueira/ContadorGamer.git

```

## 2. Entrar na pasta do projeto
```
cd contadorGamer

```

## 3. Instalar dependências
```
npm install

```

## 4. Rodar o projeto
```
npm start

```
